# 마수동 유도 인앱 바텀싯 3차 — 통합 명세

## 1. 핵심 목표

| KPI | 목표값 |
|-----|--------|
| 혜택 수령 지점 마수동 바텀싯 노출 대비 전환율 | 7% |

- 유저가 핀다포인트를 획득하는 순간(출석체크, 랜덤포인트)과 포인트함/신용관리 이탈 시점에 마수동 바텀싯을 노출
- 혜택 수령 직후의 긍정적 심리를 동의 전환으로 연결

## 2. 이벤트 스킴

| 구분 | 내용 |
|------|------|
| 대상 | 마수동(마이데이터 수동동의) 미동의 유저 전체 |
| 배포 일정 | stage: 6/10, prod: 6/15 |
| 가드레일 지표 | 노출 - "아니요" CTA 클릭률 (%) |
| 관련 작업 | 1차 인앱 바텀싯 (혜택탭/신용관리탭) 기능 종료 후 3차로 교체 |

## 3. User Flow

```
[마수동 미동의 유저]
  │
  ├─ 트리거 1: 출석체크 완료 → 포인트 획득 팝업 → [확인] CTA 클릭 또는 이탈
  ├─ 트리거 2: 랜덤포인트 획득 → 포인트 획득 팝업 → [확인] CTA 클릭 또는 이탈
  ├─ 트리거 3: 포인트함 페이지 이탈 시도 (뒤로가기/스와이프)
  └─ 트리거 4: 신용관리 페이지 이탈 시도 (뒤로가기/스와이프)
  │
  ▼
◇ 노출 정책 충족?
  ├─ NO → 바텀싯 미노출, 기존 동작 수행
  └─ YES
       │
       ▼
  [마수동 바텀싯 노출]
  - 트리거 1,2,3: "혜택과 이벤트 소식, 알려드릴까요?"
  - 트리거 4: "놓치고 있는 소식, 알려드릴까요?"
  - 서브텍스트: "지금 동의하지 않으면 일주일 동안 볼 수 없어요."
  - 동의서: 전체 동의 / 광고성 정보 수신 동의(선택) / 마케팅 목적 수집・이용 동의(선택)
  - CTA: [아니요] / [네, 알려주세요]
       │
       ├─ [아니요] 선택 → 바텀싯 닫힘 + 7일간 전 페이지 노출 금지
       ├─ 단순 이탈 (바텀싯 외 영역 터치 등) → 바텀싯 닫힘 + view 카운트 포함 + 7일 금지 미적용
       └─ [네, 알려주세요] 선택 → 동의 처리 + Snackbar 노출 + 이전 화면 유지
```

### 트리거별 미적용 조건

| 트리거 | 미적용 조건 |
|--------|------------|
| 1 (출석체크) | "포인트함 바로가기" CTA 클릭 시 / "매일 포인트 주는 출석 알림 받기" CTA 클릭 시 |
| 2 (랜덤포인트) | "포인트함 바로가기" CTA 클릭 시 |
| 3 (포인트함) | — |
| 4 (신용관리) | — |

### 출석체크 팝업 CTA 분기

| 유저 상태 | 팝업 내 CTA | 바텀싯 노출 |
|-----------|------------|------------|
| 출석체크 알림 ON | "포인트 보러가기" CTA | 클릭 시 미노출 |
| 출석체크 알림 OFF | "매일 포인트 주는 출석 알림 받기" CTA | 클릭 시 미노출 |
| 공통 | [확인] CTA 클릭 또는 이탈 | 노출 |

## 4. 화면 정책 상세

### 바텀싯 UI 구성

| 요소 | 내용 |
|------|------|
| 타이틀 (트리거 1,2,3) | "혜택과 이벤트 소식, 알려드릴까요?" |
| 타이틀 (트리거 4) | "놓치고 있는 소식, 알려드릴까요?" |
| 서브텍스트 | "지금 동의하지 않으면 일주일 동안 볼 수 없어요." |
| 동의서 - 전체 동의 | 전체 체크박스 |
| 동의서 - 항목 1 | 광고성 정보 수신 동의 (선택) + 상세 화살표 |
| 동의서 - 항목 2 | 마케팅 목적 수집・이용 동의 (선택) + 상세 화살표 |
| CTA 좌 | [아니요] (보라색 텍스트, 흰 배경, 보라 테두리) |
| CTA 우 | [네, 알려주세요] (비활성: 회색 배경 / 활성: 보라색 배경) |

### CTA 활성화 조건

- 두 동의서 모두 동의할 경우에만 [네, 알려주세요] CTA 활성화
- (선택) 동의이지만, 두 항목 모두 체크해야 CTA 활성

### 동의 완료 후

- 바텀싯 닫힘
- Snackbar 노출 (동의 완료 안내)
- 이전 화면으로 돌아감 (페이지 이동 없음)

## 5. 노출 제한 정책

| 정책 | 규칙 | 적용 범위 |
|------|------|----------|
| 일일 제한 | 어느 페이지에서든 당일 1회 노출 시 이후 전 페이지 미노출 | 전체 |
| 전체 제한 | 전체 페이지 통합 최대 5회 | 전체 |
| 페이지당 제한 | 페이지별 최대 3회 (카운트 기준: view) | 페이지별 |
| 아니요 CTA 클릭 시 | 7일간 전 페이지 노출 금지 | 전체 |
| 단순 이탈 시 | view 카운트 포함 / 7일 노출 금지 미적용 (당일 1회 정책은 적용) | 전체 |

### 적용 우선순위

```
1. 7일 노출 제한 대상인 경우 → 미노출
2. 전체 5회 노출 제한 도달 시 → 미노출
3. 해당 페이지 3회 노출 제한 도달 시 → 미노출
4. 당일 이미 노출된 경우 → 미노출
5. 위 조건에 해당하지 않을 경우 → 노출
```
